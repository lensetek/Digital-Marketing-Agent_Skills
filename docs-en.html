Digital Marketing Agent Skills

A modular autonomous digital marketing agent team with one orchestrator and 15 specialist agents. The pack uses Markdown-based instructions that can be adapted for Antigravity, Claude, Codex, Gemini custom agents, and other agent runtimes.

Install Prompt

Copy this prompt and paste it into your AI agent workspace.

download and install this agent https://github.com/lensetek/Digital-Marketing-Agent_Skills

Installer Confirmation

The installer agent must download or clone the repository, count skills/*/SKILL.md, show the source repository, install destination, detected agent count, and agent list, then ask for approval before installing.

How to Use

Start with marketing-orchestrator, provide a business brief, let it create a campaign execution plan, delegate work to specialist agents, run ethics review, and deliver an executive marketing report.

Common Use Cases

Build a 30-day marketing strategy, research competitors, plan SEO/AIO, draft landing pages, create content calendars, plan ads, design email lifecycle campaigns, analyze data, and assemble a final AI marketing agent package.

Security

Do not paste API keys, passwords, tokens, private keys, service account JSON, or raw customer data into prompts, templates, frontend files, project profiles, or public docs.

Updates and Personalization

agent-update-manager checks repository updates with confirmation. project-learning-personalization-agent stores safe project-level learning only.
