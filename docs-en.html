Digital Marketing Agent Skills Docs

A modular digital marketing agent team with one orchestrator and 21 specialist agents. The pack uses Markdown-based instructions that can be adapted for Antigravity, Claude, Codex, Gemini custom agents, and other agent runtimes.

Install Prompt

download and install this agent https://github.com/lensetek/Digital-Marketing-Agent_Skills

Security Rule

Do not paste API keys, passwords, tokens, private keys, service account JSON, billing credentials, or raw customer data into prompts, templates, frontend files, or public docs. Treat VITE_*, NEXT_PUBLIC_*, browser JavaScript, HTML, and public assets as public.
