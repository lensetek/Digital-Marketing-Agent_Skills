## English

Digital Marketing Agent Skills is a portable AI marketing team with **21 professional agent skills**. Users work through `marketing-orchestrator`, which receives the brief, plans the work, coordinates specialists, manages approval gates, and delivers the final marketing report.

## Quick Install Prompt

Paste this prompt into Antigravity, Claude, Codex, Gemini/custom agent, or any AI agent workspace that can download and install Markdown-based agent instructions.

```text
download and install this agent https://github.com/lensetek/Digital-Marketing-Agent_Skills
```

The installer agent must confirm the repository, destination, detected agent count, and agent list before installing. Current expected inventory: **21 agent skills** plus **14 reusable capabilities** in `capabilities/`.

## Landing Page

Open [index.html](index.html) to view the static landing page with a copyable install prompt box.

| Agent skill | Function |

| `marketing-orchestrator` | Main entry point that receives briefs, plans workflows, routes work to specialists, manages approvals, and compiles the final report. |
| `marketing-director` | Defines marketing strategy, positioning, target segments, channel priorities, budgets, and campaign direction. |
| `brand-strategist` | Develops brand positioning, value propositions, messaging frameworks, brand voice, and differentiation. |
| `market-research-analyst` | Researches markets, trends, customer needs, competitors, and growth opportunities. |
| `competitive-intelligence-agent` | Monitors competitor offers, pricing, campaigns, SEO activity, and industry benchmarks. |
| `customer-journey-funnel-strategist` | Maps customer journeys, designs funnels and touchpoints, and identifies conversion improvements. |
| `seo-aio-strategist` | Plans keyword research, topic clusters, semantic SEO, AI search optimization, and organic content opportunities. |
| `paid-ads-specialist` | Plans Google, Meta, and TikTok advertising, including targeting, budget allocation, creative angles, and ad copy. |
| `influencer-partnership-strategist` | Plans influencer selection, affiliate programs, partnerships, sponsorships, and collaboration campaigns. |
| `content-strategist` | Creates content pillars, editorial calendars, campaign narratives, and channel-specific content plans. |
| `growth-creative-agent` | Generates unconventional campaign ideas, viral concepts, meme marketing, hooks, and creative experiments. |
| `video-creative-director` | Produces video concepts, scripts, storyboards, hooks, and creative direction for YouTube, TikTok, and Reels. |
| `video-pattern-reverse-engineer` | Analyzes reference videos to extract hooks, structures, pacing, CTAs, and reusable creative patterns. |
| `social-media-engagement-agent` | Plans captions, social calendars, community engagement, audience interaction, and organic growth. |
| `landing-page-conversion-specialist` | Designs landing-page structure, conversion copy, calls to action, and CRO recommendations. |

| `agent-update-manager` | Checks repository changes, reviews changelogs, assesses compatibility, and proposes approval-gated updates. |

## Practical Use Cases

- **New product launch:** build positioning, research the market, map the funnel, prepare campaign content and ads, then define measurement and compliance checks.
- **Integrated campaign planning:** turn one business brief into coordinated SEO, paid media, social, influencer, email, landing-page, and retention plans.
- **Content engine:** create content pillars, an editorial calendar, social posts, video scripts, hooks, and reusable creative patterns for multiple channels.
- **Lead generation and conversion:** design ads, landing pages, lead-nurture automation, CRM segmentation, and conversion optimization as one connected funnel.
- **Retention and repeat sales:** develop lifecycle email, loyalty, win-back, churn-prevention, and personalized CRM campaigns.
- **Competitor and market intelligence:** compare competitor positioning, pricing, campaigns, SEO, and customer opportunities to support strategic decisions.
- **Campaign audit and optimization:** analyze KPIs and funnel performance, identify weak points, recommend experiments, and review privacy, claims, and platform risk.
- **Local business marketing:** produce a practical campaign plan for restaurants, clinics, schools, property businesses, professional services, or other local brands.

## Capability Layer

Technical helpers live in `capabilities/`: browser automation, web research, scraping, video understanding, X/Twitter workflow, GitHub, file, document, spreadsheet, presentation, email, scheduling, knowledge base/RAG, and project memory. These are helper capabilities, not extra marketing agents.

## Security

Do not paste API keys, passwords, tokens, private keys, service account JSON, or raw customer-sensitive data into prompts, templates, frontend files, or public docs.

Treat `VITE_*`, `NEXT_PUBLIC_*`, browser JavaScript, HTML, and public assets as public. See [docs/security-checklist.md](docs/security-checklist.md) before adding integrations, automation, APIs, or deployment examples.

## Bahasa Indonesia

Digital Marketing Agent Skills adalah tim marketing AI portabel dengan **21 agent skill profesional**. User cukup berinteraksi dengan `marketing-orchestrator`, yang menerima brief, menyusun rencana kerja, mengoordinasikan specialist, mengelola approval gate, dan menghasilkan laporan marketing final.

Paket ini menggunakan instruksi berbasis Markdown sehingga dapat diadaptasi untuk Antigravity, Claude, Codex, Gemini/custom agent, dan runtime agent lain yang mendukung instruksi berbasis role atau project.

### Prompt Instalasi

Salin prompt berikut ke workspace AI agent Anda:

```text
download and install this agent https://github.com/lensetek/Digital-Marketing-Agent_Skills
```

Installer harus mengonfirmasi repository, lokasi instalasi, jumlah agent yang terdeteksi, dan daftar agent sebelum memasang. Inventaris yang diharapkan: **21 agent skill** dan **14 reusable capability** di `capabilities/`.

### Cara Kerja

1. Berikan brief bisnis kepada `marketing-orchestrator`.
2. Orchestrator membuat rencana dan memilih specialist yang relevan.
3. Agent strategy dan research dijalankan sebelum eksekusi channel.
4. Agent acquisition, content, conversion, retention, dan automation bekerja sesuai objective.
5. `analytics-optimization-analyst` menyiapkan measurement plan.
6. `ethics-compliance-reviewer` memeriksa compliance, privacy, claim, dan risiko reputasi.
7. Orchestrator menyatukan hasil menjadi laporan marketing final.

### Use Case Praktis

- **Peluncuran produk baru:** menyusun positioning, riset pasar, funnel, campaign content, ads, measurement, dan compliance check.
- **Perencanaan campaign terintegrasi:** mengubah satu brief bisnis menjadi rencana SEO, paid media, social media, influencer, email, landing page, dan retention yang selaras.

- **Retention dan repeat sales:** membangun lifecycle email, loyalty, win-back, churn prevention, dan campaign CRM yang lebih relevan.
- **Market dan competitor intelligence:** membandingkan positioning, harga, campaign, SEO, serta peluang customer untuk mendukung keputusan strategi.
- **Audit dan optimasi campaign:** menganalisis KPI dan funnel, menemukan titik lemah, merekomendasikan eksperimen, serta memeriksa privacy, claim, dan risiko platform.
- **Marketing bisnis lokal:** membuat campaign praktis untuk restoran, klinik, sekolah, properti, jasa profesional, dan brand lokal lainnya.

### Capability Layer

Folder [capabilities/](capabilities/) berisi 14 kemampuan reusable untuk browser automation, web research, scraping, video understanding, X/Twitter workflow, GitHub, file, document, spreadsheet, presentation, email, scheduling, knowledge base/RAG, dan project memory. Capability ini merupakan helper lintas-agent dan tidak dihitung sebagai agent marketing tambahan.

Utamakan script, API, CLI, MCP, connector, atau browser automation. Gunakan Computer Use hanya sebagai fallback ketika tidak ada jalur otomasi yang lebih aman.

### Security First

Jangan memasukkan API key, password, token, private key, service account JSON, billing credential, atau raw customer-sensitive data ke prompt, template, file frontend, maupun dokumentasi publik.

Semua konfigurasi frontend seperti `VITE_*`, `NEXT_PUBLIC_*`, JavaScript browser, HTML, dan public assets harus dianggap publik. Lihat [docs/security-checklist.md](docs/security-checklist.md) sebelum menambahkan integrasi, automation, API, atau contoh deployment.
