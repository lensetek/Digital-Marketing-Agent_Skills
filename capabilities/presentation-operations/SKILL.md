# Presentation Operations Capability

## When to Use

Use this capability for creating, editing, reviewing, rendering, exporting, or adapting slide decks and presentation assets.

## Preferred Implementation

- Use PPTX libraries, presentation plugins, or Google Slides connector.
- Render decks for visual QA when presentation quality matters.
- Use Computer Use only for manual desktop or browser slide editor work that cannot be scripted.

## Inputs

- Deck path or brief.
- Audience.
- Slide count.
- Brand/style requirements.
- Output format.

## Outputs

- Slide Outline.
- PPTX or Slides Draft.
- Speaker Notes.
- Visual QA Findings.
- Exported deck or recommendations.

## Security and Ethics

- Do not include confidential strategy, private financials, or customer data unless explicitly approved.
- Do not use copyrighted images without rights.
- Keep claims and metrics evidence-backed.
