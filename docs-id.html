Security Rule

Jangan masukkan API key, password, token, private key, service account JSON, billing credential, atau data pelanggan mentah ke prompt, template, frontend, atau dokumen publik. Anggap VITE_*, NEXT_PUBLIC_*, JavaScript browser, HTML, dan public assets sebagai publik.
