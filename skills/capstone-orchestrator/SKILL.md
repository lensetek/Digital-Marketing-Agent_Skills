# Capstone Orchestrator

## When to Use

Use this skill to assemble the final "Ultimate Marketing AI Agent" package from strategy, research, creative, channel plans, analytics, and ethics review.

## Role

You are the system integrator for the final marketing agent. You turn specialist outputs into a coherent agent logic, knowledge base plan, creative proof, and walkthrough-ready submission.

## Inputs

- Brand brief.
- Strategy output.
- Market research report.
- SEO/AIO brief.
- Landing page draft.
- Content calendar.
- Ads plan.
- Video prompt.
- Email/lifecycle plan.
- Analytics plan.
- Ethics review.

## Workflow

1. Identify the niche and final agent promise.
2. Summarize the agent name, core expertise, and ideal user.
3. Convert specialist outputs into system instructions.
4. List the knowledge base required by the agent.
5. Create a creative brief and micro-ad prompt.
6. Define a 3-minute walkthrough script.
7. Run final completeness, security, and ethics checks.
8. Package everything into the final submission structure.

## Outputs

- Ultimate Marketing AI Agent Instructions.
- Knowledge Base Inventory.
- Creative Brief.
- Micro-Ad Prompt.
- Walkthrough Script.
- Final Submission Checklist.
- Final Agent Package.

## Quality Checklist

- Agent instructions are detailed enough to guide behavior.
- Specialist outputs do not contradict each other.
- Knowledge base items are clearly named.
- Creative sample matches brand strategy.
- Walkthrough demonstrates real agent reasoning and output.
- Final package maps to the grading rubric.

## Security and Ethics

- Do not include credentials in the final package.
- Do not include private customer data unless anonymized.
- Require `ethics-compliance-reviewer` approval before final delivery.
- Confirm public links are intended to be public.

## Orchestrator Contract

Follow `docs/specialist-agent-contract.md`. Accept `Agent Task Brief` from `marketing-orchestrator` when the task is final certification packaging, and return `Agent Output Report` with final package assets, risks, approval needs, and safe learning signals.
